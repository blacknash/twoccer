Twoocer
=======

Twoccer is an experiment to recollect text from Twitter related with the WorldCup 2014.

I made this project to learn more about : 
* Sentiment Analysis
* Text Classification
* Machine Learnings
* GoLang
* Data Analysis
* Enjoy de Worldcup :-)
